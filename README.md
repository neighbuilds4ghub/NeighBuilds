# Neighbuilds

Neighbuilds is a personal project site for sharing 3D scans, digital objects, project files, notes, and other useful work.

The goal is to make these files available for personal use in a simple value-for-value model: use what helps you, share what you build, and support the work if you find it useful.

## Website

The live site will be available at:

https://www.neighbuilds.com

## What this site will include

- 3D scans
- Downloadable model files
- Project notes
- Build documentation
- Reference images
- Experiments and prototypes

## Use policy

You're welcome to use any you see here for personal or educational use. If you do find something useful or just want to support my work, throw me a dollar!
